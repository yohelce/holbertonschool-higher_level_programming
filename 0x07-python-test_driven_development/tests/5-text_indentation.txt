The ``5-text_indentation`` module

Using ``text-indentation``
---------------------------

Import the function:

	>>> text_indentation = __import__('5-text_indentation').text_indentation

Now test it:

	>>> text_indentation("Hello.")

	<BLANKLINE>

	>>> text_indentation("Hello. World: Here I am!")
	Hello.
	<BLANKLINE>
	World:
	<BLANKLINE>
	Here I am!

	>>> text_indentation(".    :     ?  ")
	.
	<BLANKLINE>
	:
	<BLANKLINE>
	?
	<BLANKLINE>

	>>> text_indentation("")
